# Contributing to Open Source with PyMavs ### About This short tutorial is geared towards those who have not used Github before. By the end of this tutorial you will be able to clone, commit, and push using [PyMavs website](https://pymavs.github.io/). Github Link: [PyMavs Website](https://github.com/PyMavs/pymavs.github.io) #### Note Before continuing make sure that git is installed on your machine!! There are plenty of resources out there such as... - [This Simple Blog](https://gist.github.com/derhuerst/1b15ff4652a867391f03) - [or This 20 Minuet Video](https://www.youtube.com/watch?v=J_Clau1bYco) While both are helpful tutorials, you might run into some problems. In otherwords, we are creating a copy of the [Websites Code](https://github.com/PyMavs/pymavs.github.io) onto your own Github account, allowing you to freely experiement with the code. With a fork you can change anything you want on your copy and when you are ready you can submit a pull request. ![Image of Fork Button](http://readme-pics.s3.amazonaws.com/fork_button.jpg)
